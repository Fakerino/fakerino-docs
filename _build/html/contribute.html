Contribute¶

Fakerino needs you.

Every contribution is really welcome.

Contribute to Fakerino code¶

Easy step for contribute:

  • Fork the project

  • Branch from master with a descriptive name for your branch, example:
    • feat/name-of-feature
    • fix/bug-fix
    • issue/issue-number

  • Write unit tests and make sure all tests of the entire project will pass.

  • Insert documentation on code, licence and author note and check the code style(PSR-2) with tools like phpcs

  • Make a pull request

Contribute to Fakerino data¶

You could also contribute to increase the Fakerino supported data via the ODS project.
